## v0.6 — Анонимизация лиц (Hide Faces)

> Двухэтапный воркфлоу: сначала автоматический поиск всех лиц с возможностью ручной коррекции,
> затем применение эффекта (размытие / пикселизация / заливка / эмодзи / стикер).
> **Приоритет — максимальная полнота детекции (recall) над скоростью.** Хорошая работа на
> групповых фото (свадьбы, классы, концерты, толпы).

### Модели (ансамбль для max recall)

- [ ] Интеграция SCRFD-10G-KPS (основной детектор)
- [ ] Интеграция RetinaFace-R50 (профили, occlusion)
- [ ] Интеграция YOLOv8-face (плотные группы)
- [ ] Интеграция YuNet (мелкие лица 16×16+)
- [ ] Реестр моделей с метаданными (size, quality, speed)

### Пайплайн детекции (`thoroughDetect`)

- [ ] Multi-scale инференс на 7 шкалах (512/768/1024/1280/1600/2048/2560)
- [ ] Test-Time Augmentation (horizontal flip)
- [ ] Tile-based детекция (1024×1024, overlap 384) для крупных фото
- [ ] Weighted Box Fusion для объединения результатов ансамбля
- [ ] Soft-NMS как fallback для одиночной модели
- [ ] Адаптивный пайплайн (smartDetect) с авто-выбором стратегии

### Опциональные техники максимального recall

- [ ] Image pyramid с шагом 0.83×
- [ ] ROI refinement в плотных областях (повторный zoom-in)
- [ ] Детекция тел (YOLOv8-person) для подсказок «возможно пропущено лицо»
- [ ] Landmark-refinement для отсева false positives

### UI и UX

- [ ] Двухэтапный wizard (Детекция → Применение эффекта)
- [ ] 5 пресетов: Быстрый / Стандартный / Тщательный / Максимум / Параноидальный
- [ ] Прогресс-бар с этапом, временем и счётчиком найденных лиц

- [ ] Bulk-операции (выбрать все/по порогу, lasso-selection)
