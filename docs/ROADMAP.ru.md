## v0.6 — Анонимизация лиц (Hide Faces)

> Двухэтапный воркфлоу: сначала автоматический поиск всех лиц с возможностью ручной коррекции,
> затем применение эффекта (размытие / пикселизация / заливка / эмодзи / стикер).
> **Приоритет — максимальная полнота детекции (recall) над скоростью.** Хорошая работа на
> групповых фото (свадьбы, классы, концерты, толпы).

### Модели (ансамбль для max recall)

- [x] Интеграция SCRFD-10G-KPS (основной детектор)
- [x] Интеграция RetinaFace-R50 (профили, occlusion)
- [x] Интеграция YOLOv8-face (плотные группы)
- [x] Интеграция YuNet (мелкие лица 16×16+)
- [x] Реестр моделей с метаданными (size, quality, speed)

### Пайплайн детекции (`thoroughDetect`)

- [x] Multi-scale инференс на 7 шкалах (512/768/1024/1280/1600/2048/2560)
- [x] Test-Time Augmentation (horizontal flip)
- [x] Tile-based детекция (1024×1024, overlap 384) для крупных фото
- [x] Weighted Box Fusion для объединения результатов ансамбля
- [x] Soft-NMS как fallback для одиночной модели
- [x] Адаптивный пайплайн (smartDetect) с авто-выбором стратегии

### Опциональные техники максимального recall

- [x] Image pyramid с шагом 0.83×
- [x] ROI refinement в плотных областях (повторный zoom-in)
- [x] Детекция тел (YOLOv8-person) для подсказок «возможно пропущено лицо»
- [x] Landmark-refinement для отсева false positives

### UI и UX

- [x] Двухэтапный wizard (Детекция → Применение эффекта)
- [x] 5 пресетов: Быстрый / Стандартный / Тщательный / Максимум / Параноидальный
- [x] Прогресс-бар с этапом, временем и счётчиком найденных лиц
- [x] Кнопка «Стоп с сохранением промежуточного результата» (AbortSignal)
- [x] Live-превью боксов по мере детекции
- [x] Цветовая индикация уверенности (votes: 1=жёлтый, 2=синий, 3-4=зелёный)
- [x] Миниатюры найденных лиц (face crops grid) для быстрого ревью
- [x] Bulk-операции (выбрать все/по порогу, lasso-selection)
