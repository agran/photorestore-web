# Безопасность

> 🇬🇧 [English version](./SECURITY.md)

## Гарантия конфиденциальности

**PhotoRestore Web обрабатывает все изображения 100% локально в вашем браузере.**

- Никакие изображения не загружаются ни на какой сервер.
- Никакой телеметрии или аналитики не собирается.
- Никаких аккаунтов или аутентификации не требуется.
- Никаких сторонних скриптов отслеживания не включено.

## Заголовки COOP / COEP

Для включения `SharedArrayBuffer` (многопоточность WASM) приложение устанавливает:

```
Cross-Origin-Opener-Policy: same-origin
Cross-Origin-Embedder-Policy: require-corp
```

Эти заголовки устанавливаются:

1. В `vite.config.ts` (сервер разработки)
2. В `public/coi-serviceworker.js` (продакшн, через Service Worker)

## Загрузка моделей

Модели загружаются с HuggingFace и верифицируются по SHA-256 контрольным суммам. Загруженные модели хранятся в Cache API браузера и никогда не покидают устройство.
